# Student Management System

A CRUD-based Student Management System built using ASP.NET Core MVC.

## Features

- Add Student
- Edit Student
- Delete Student
- Form Validation
- Repository Pattern
- Dependency Injection
- Entity Framework Core
- SQL Server

## Technologies Used

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap 5

## How to Run

1. Clone the repository
2. Update connection string in appsettings.json
3. Run Update-Database
4. Press F5
